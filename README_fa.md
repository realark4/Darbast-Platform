## 🌐 معرفی
**دربست (DARBast)** یک موتور پیشرفته مدیریت املاک و مستغلات است که با استفاده از **FastAPI** و **SQLAlchemy** توسعه یافته است. این سیستم راهکاری جامع، مقیاس‌پذیر و کاملاً ساختاریافته برای ثبت آگهی، مدیریت کاربران و نظارت ادمین ارائه می‌دهد. تمرکز اصلی این پلتفرم بر امنیت بالا، احراز هویت مدرن، مدیریت سطح دسترسی (RBAC) و یک چرخه منظم برای تایید آگهی‌ها می‌باشد.

---

## 🛠 ویژگی‌ها و قابلیت‌های کلیدی

### ۱. 🔐 سیستم احراز هویت پیشرفته
- **ثبت‌نام امن:** رمزهای عبور با استفاده از الگوریتم استاندارد `Bcrypt` هش می‌شوند.
- **ورود ترکیبی (Hybrid Login):** پشتیبانی همزمان از توکن‌های **JWT Bearer** و **کوکی‌های امن HTTP-only** برای تضمین حداکثر امنیت.
- **نشست ماندگار:** قابلیت «مرا به خاطر بسپار» با ماندگاری ۷ روزه.
- **مدیریت سطح دسترسی (RBAC):** سطوح دسترسی سلسله‌مراتبی شامل نقش‌های `Admin`، `Employee`، `Agent` و `User`.
- **سیستم ضد هک (Anti-Brute Force):** قفل شدن خودکار IP کاربر پس از چند تلاش ناموفق متوالی.

### ۲. 🏠 موتور مدیریت املاک
- **عملیات کامل CRUD:** ایجاد، مشاهده، ویرایش و حذف آگهی‌ها با اعتبارسنجی دقیق مالکیت.
- **چرخه نظارت ادمین:** سیستم وضعیت سه مرحله‌ای برای آگهی‌ها (`در انتظار تایید`، `تایید شده`، `رد شده`).
- **فیلترهای پیشرفته:** جستجوی املاک بر اساس محله، محدوده قیمت و نوع معامله (`خرید`، `فروش`، `رهن`، `اجاره`).
- **گالری تصاویر:** سیستم قدرتمند برای آپلود چندین تصویر برای یک ملک و سرویس‌دهی امن آن‌ها.

### ۳. 👤 تجربه کاربری اختصاصی
- **داشبورد آگهی‌های من (`My Ads`):** پنل اختصاصی کاربر برای مدیریت آگهی‌های ثبت شده خود.
- **نشان‌شده‌ها (Bookmarks):** ذخیره‌سازی آسان املاک مورد علاقه برای دسترسی‌های بعدی.
- **مدیریت پروفایل:** مسیرهای اختصاصی (`/me`) جهت مشاهده و ویرایش اطلاعات کاربری.

### ۴. 🛠 پنل مدیریت حرفه‌ای (SQLAdmin)
- **طراحی اختصاصی:** رابط کاربری کاملاً بازطراحی شده با تم تیره (OLED) و المان‌های بصری مدرن SVG.
- **عملیات گروهی (Bulk Actions):** تایید، رد یا مسدودسازی حساب‌ها به صورت دسته‌جمعی.
- **نمایش‌های بصری:** استفاده از Badgeهای وضعیت پویا و پیش‌نمایش شناور تصاویر (Hover Preview) در پنل مدیریت.

## 📂 ساختار پروژه

```text
DARBast-Platform/
├── Api/               # مسیرها و نقاط اتصال API
├── Auth/              # توابع احراز هویت، JWT و مدیریت دسترسی
├── Core/              # تنظیمات اصلی، متغیرهای محیطی و راه‌اندازی FastAPI
├── Db/                # مدل‌های دیتابیس (SQLAlchemy) و مدیریت نشست‌ها
├── Migration/         # اسکریپت‌های مایگریشن با Alembic
├── Schemas/           # مدل‌های Pydantic جهت اعتبارسنجی ورودی/خروجی
├── admin_static/      # قالب‌های اختصاصی تغییریافته برای SQLAdmin
├── alembic.ini        # تنظیمات Alembic
└── requirements.txt   # پیش‌نیازها و وابستگی‌های پایتون
```

## 🚀 نقاط اتصال مهم (API Endpoints)

### بخش احراز هویت (`/auth`)
- `POST /auth/register` : ثبت‌نام کاربر جدید
- `POST /auth/login` : احراز هویت و دریافت توکن + کوکی امن
- `POST /auth/logout` : خروج و اتمام امن نشست
- `GET /auth/me` : دریافت اطلاعات پروفایل جاری

### بخش آگهی‌ها (`/advertisement` و `/my-ads`)
- `GET /advertisement/` : دریافت تمام آگهی‌های تاییدشده (پشتیبانی از فیلتر)
- `GET /advertisement/{id}` : مشاهده جزئیات یک آگهی خاص
- `POST /my-ads/` : ثبت یک آگهی جدید توسط کاربر
- `POST /my-ads/{id}/images` : آپلود تصاویر برای یک آگهی
- `GET /bookmarks/` : مشاهده لیست آگهی‌های نشان‌شده کاربر

## 💻 نصب و راه‌اندازی سریع

### ۱. دریافت سورس کد
```bash
git clone https://github.com/your-username/DARBast-Platform.git
cd DARBast-Platform
```

### ۲. ساخت محیط مجازی (Virtual Environment)
```bash
# ساخت محیط
python -m venv .venv

# فعال‌سازی در ویندوز:
.venv\Scripts\activate
# فعال‌سازی در لینوکس و مک:
source .venv/bin/activate
```

### ۳. نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### ۴. تنظیم متغیرهای محیطی
یک فایل `.env` در مسیر اصلی پروژه با محتوای زیر بسازید:
```env
SECRET_KEY="یک_کلید_بسیار_امن_و_طولانی"
ACCESS_TOKEN_EXPIRE_MINUTES=10080
ADMIN_MAX_LOGIN_ATTEMPTS=5
```

### ۵. آماده‌سازی دیتابیس (Migration)
آخرین تغییرات دیتابیس را روی دیتابیس اعمال کنید:
```bash
alembic upgrade head
```

### ۶. اجرای سرور
برنامه را با uvicorn اجرا کنید:
```bash
uvicorn Core.Core:app --reload
```
حالا می‌توانید مستندات تعاملی API را در آدرس زیر مشاهده کنید: `http://localhost:8000/docs`
