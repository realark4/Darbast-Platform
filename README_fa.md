### 🌐 معرفی
پروژه «دربست» یک موتور پیشرفته مدیریت املاک و مستغلات است که با FastAPI توسعه یافته است. این سیستم راهکاری جامع برای ثبت آگهی، مدیریت کاربران و نظارت ادمین با تمرکز بر امنیت و تجربه کاربری سطح بالا ارائه می‌دهد.

---

## 🛠 ویژگی‌ها و قابلیت‌ها

### ۱. 🔐 سیستم احراز هویت
| قابلیت | توضیحات |
| :--- | :--- |
| **ثبت‌نام** | ثبت‌نام امن کاربر با هش کردن رمز عبور. |
| **ورود ترکیبی** | پشتیبانی همزمان از توکن‌های JWT و کوکی‌های امن. |
| **نشست ماندگار** | قابلیت «مرا به خاطر بسپار» با ماندگاری ۷ روزه. |
| **خروج** | خروج کامل از سیستم و حذف کوکی‌های امنیتی. |
| **RBAC** | مدیریت دسترسی بر اساس نقش: ادمین، کارمند، مشاور، کاربر. |

### ۲. 🏠 سیستم املاک
| قابلیت | توضیحات |
| :--- | :--- |
| **عملیات CRUD** | مدیریت کامل آگهی‌ها (ایجاد، مشاهده، ویرایش، حذف) با تایید مالکیت. |
| **چرخه سه‌مرحله‌ای** | چرخه نظارت سه مرحله‌ای: معلق، تایید شده، رد شده. |
| **فیلتر پیشرفته** | فیلتر پیشرفته بر اساس محله، محدوده قیمت و نوع معامله. |
| **انواع معامله** | پشتیبانی از انواع معامله: خرید، فروش، رهن و اجاره. |
| **گالری تصاویر** | پشتیبانی از آپلود چندین تصویر و سرویس‌دهی فایل‌های استاتیک. |

### ۳. 👤 تجربه کاربری
| قابلیت | توضیحات |
| :--- | :--- |
| **آگهی‌های من** | داشبورد اختصاصی برای مدیریت آگهی‌های شخصی. |
| **نشان‌شده‌ها** | سیستم نشان کردن (علاقه‌مندی‌ها) برای دسترسی سریع. |
| **مدیریت پروفایل** | مشاهده و مدیریت اطلاعات پروفایل کاربری. |

### ۴. 🛠 پنل مدیریت حرفه‌ای
| قابلیت | توضیحات |
| :--- | :--- |
| **قفل امنیتی** | سیستم ضد هک: قفل شدن IP پس از تلاش‌های ناموفق متوالی. |
| **عملیات گروهی** | عملیات دسته‌جمعی: تایید، مسدودسازی یا ارتقای نقش کاربران. |
| **داشبورد اختصاصی** | داشبورد اختصاصی با المان‌های SVG و طراحی تیره مدرن. |
| **فرمت‌دهی پویا** | نمایش بصری وضعیت‌ها با Badge و پیش‌نمایش شناور تصاویر. |

## 🚀 نقاط اتصال (API)

### احراز هویت و کاربر
- `POST /auth/register` : ثبت‌نام کاربر
- `POST /auth/login` : ورود امن (توکن + کوکی)
- `POST /auth/logout` : پایان نشست
- `GET /auth/me` : پروفایل فعلی

### آگهی‌ها
- `GET /advertisement/` : لیست عمومی (با فیلترها)
- `GET /advertisement/{id}` : نمایش جزئیات
- `POST /my-ads/` : ثبت آگهی جدید
- `POST /my-ads/{id}/images` : آپلود تصاویر ملک
- `GET /bookmarks/` : مشاهده املاک ذخیره‌شده

## 🛠 پشته تکنولوژی
- **بک‌اند**: FastAPI (پایتون 3.10+)
- **ORM**: SQLAlchemy 2.0
- **امنیت**: Passlib (Bcrypt), Python-Jose (JWT)
- **رابط کاربری ادمین**: SQLAdmin (شخصی‌سازی شده)
- **پایگاه داده**: SQLite (طرح‌واره آماده برای تولید)
- **تنظیمات**: Pydantic Settings (مبتنی بر محیط)

## 💻 نصب و راه‌اندازی

۱. **راه‌اندازی محیط**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # در ویندوز: .\.venv\Scripts\activate
   pip install -r requirements.txt
   ```

۲. **پیکربندی**:
   یک فایل `.env` ایجاد کنید:
   ```env
   SECRET_KEY="your_secret"
   ACCESS_TOKEN_EXPIRE_MINUTES=10080
   ADMIN_MAX_LOGIN_ATTEMPTS=5
   ```

۳. **اجرای برنامه**:
   ```bash
   uvicorn Core.Core:app --reload
   ```
